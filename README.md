# VIT Chennai Club Management System (ClubHub)

A centralized web-based platform for managing student clubs, events, memberships, and announcements.

## 🚀 Overview
ClubHub is designed to streamline the club operations at VIT Chennai. It provides a modern, responsive, and role-based interface for Students, Club Heads, and Administrators.

## ✨ Key Features
- **Centralized Dashboard**: Personalized stats and activities for all users.
- **Role-Based Access**: 
  - **Students**: Explore clubs, join memberships, register for events.
  - **Club Heads**: Manage members, create events, post announcements.
  - **Admins**: Approve events, manage all clubs, oversee system stats.
- **Event Lifecycle**: Creation -> Approval -> Registration (with deadlines) -> Attendance.
- **Announcements**: Pinned and categorized updates from various clubs.
- **Dark/Light Mode**: Premium UI with glassmorphism and modern aesthetics.
- **Analytics**: Visualization of membership trends and event distributions using Recharts.

## 🛠️ Tech Stack
- **Frontend**: React.js (Vite), React Router, React Icons.
- **Styling**: Vanilla CSS with Design Tokens & Glassmorphism.
- **Charts**: Recharts for data visualization.
- **State Management**: React Context (Auth, Theme).

## 📁 Project Structure
- `/src/context`: Auth and Theme context providers.
- `/src/pages`: Main application pages (Home, Dashboard, Clubs, Events, etc.).
- `/src/components`: Reusable layout and UI components.
- `/src/data`: Mock data for development and demonstration.
- `/src/index.css`: Global design system and theme tokens.

## ⚡ Quick Start
1. `npm install`
2. `npm run dev`
